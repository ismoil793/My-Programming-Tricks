200: “Everything is OK.”
201: “Created.”
202: “Accepted.” 
204: “No Content.” - Server processed the request, but did not return any content

400: “Bad Request.” - Client side issue
401: “Unauthorized” or “Authorization Required.” 
402: “Payment Required.” - e.g. You haven’t paid your Shopify fees
403: “Access to that resource is forbidden.”
404: “The requested resource was not found.” 
410 – Gone, like 404, but it returns that, page requested was removed
429: “Too many requests.”

500: “There was an error on the server and the request could not be completed.”
501: “Not Implemented.” server does not support the functionality required by request
502: “Bad Gateway.” means that one server has received an invalid response from another
503: “The server is unavailable to handle this request right now.”
